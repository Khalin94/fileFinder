사용자 신청을 위해 로그인 화면에서 [사용자등록신청] 버튼을 선택합니다.
 
 
  사용자를 확인하기 위해 주민등록번호를 입력합니다.
 
 
  주민등록번호 및 성별 확인 후 개인정보를 입력합니다.
*표시는 필수항목이므로 반드시 입력하여야 합니다.
 
 
  사용자 등록정보를 확인하고 [확인]버튼을 선택합니다.
[인쇄]버튼을 선택하여 [의정자료 전자유통 시스템 사용자등록 신청서]를 인쇄합니다.
 
 
 
  인쇄된 [별지서식1호] 에 기관장 직인 또는 관인을 찍어 국회(02-788-3378)로 FAX로 발송합니다.
  국회에서 시스템 사용 승인이 완료되어야 시스템 사용이 가능합니다. 로그인 화면에서 [사용자등록확인] 버튼을 선택하여 사용자로 등록 되었는지 확인할 수 있습니다.
 
 
  시스템 이용약관을 확인 후 동의 여부를 결정합니다.
 
 
  사용자 확인을 위해 주민등록번호를 입력합니다.
 
 
  인증서 선택창에서 인증서를 선택하고 비밀번호를 입력합니다.
반드시 귀기관의 RA(인증등록)시스템에서 발급받은 본인의 GPKI인증서를 등록해야 합니다.
 
 
  개인정보 확인 후 추가사항을 입력합니다.
*표시는 필수 항목이므로 반드시 입력하여야 합니다..
 
  [확인] 버튼을 선택하면 사용자 등록이 완료됩니다.
